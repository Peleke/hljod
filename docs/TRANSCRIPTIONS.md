## How to Transcribe

1. Listen to the original track
2. Identify key, tempo, and main melody/bass notes
3. Write in strudel mini-notation
4. Test on strudel.cc
5. Compare to original — does it sound RIGHT, not just close?
6. Update the lesson editor and change status to ACCURATE
